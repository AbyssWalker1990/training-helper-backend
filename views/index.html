API usage examples

Register new user:

POST

https://large-periodic-guan.glitch.me/register

{ "user": "your_username", "password": "your_password" }

Authenticate user and get Access Token:

POST

https://large-periodic-guan.glitch.me/auth

{ "user": "your_username", "password": "your_password" }

Get new Access Token using Refresh Token stored in cookie:

GET

https://large-periodic-guan.glitch.me/refresh

Logout and delete Refresh Token from cookie and database (still need to delete Access Token):

GET

https://large-periodic-guan.glitch.me/logout
